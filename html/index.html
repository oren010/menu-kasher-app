ü§ñ Claude Configuration

To authenticate Claude CLI:

  1. Once connected to VS Code, open a new terminal.
  2. Type the command: claude and press Enter.
  3. You will be prompted to sign in to your Claude account.
  4. If a popup appears with a link, close it.
  5. Instead, copy the link shown in the terminal and paste it into a new browser tab.
  6. Authorize access in the browser, then copy the password displayed on the success page.
  7. Paste the password into your terminal and press Enter to complete the connection.

üåê Custom Domain Configuration

By default, the prompt will configure your app with the Elestio-provided CNAME (vibecoder-0ocus-u12869.vm.elestio.app).

If you want to use a custom domain, open the .env file and modify the DOMAIN variable by adding a comma followed by your custom CNAME.

For example, if your variable looks like this:

DOMAIN=vibecoder-0ocus-u12869.vm.elestio.app

Change it to:

DOMAIN=vibecoder-0ocus-u12869.vm.elestio.app,your-custom-cname
